# Numerology Monorepo (Mobile Friendly)

## IMPORTANT (Mobile GitHub Upload)
If your phone cannot upload folders:

### Option A (Recommended)
1. Upload this ZIP as-is to GitHub
2. GitHub automatically preserves folders

### Option B (Manual)
1. Create files one by one
2. Prefix folder names:
   numerology-api__index.js
   numerology-angular-ui__package.json

(Option A is strongly recommended)

## Render Deployment
- Web Service → Root Directory: numerology-api
- Static Site → Root Directory: numerology-angular-ui
